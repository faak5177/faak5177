# Модуль 3. Диаграмма прецедентов (UseCase)

**Исполнитель:** Соловьёв Георгий Александрович 
**Предметная область:** Магазин строительных материалов

## Исходный файл

Диаграмма: `docs/UseCase_Соловьёв.drawio`

## Как открыть и экспортировать в PDF

1. Открыть [app.diagrams.net](https://app.diagrams.net) (он же draw.io).
2. File → Open from → Device... → выбрать `UseCase_Соловьёв.drawio`.
3. (При необходимости) подправить положения, добавить рамки.
4. File → Export as → PDF... → `Модуль3_UseCase_Соловьёв.pdf`.
5. Залить PDF в `docs/` через GitHub UI.

## Акторы (4)

1. **Гость** — неавторизованный посетитель. Просмотр каталога без возможности покупки.
2. **Авторизированный клиент** — зарегистрированный покупатель.
3. **Менеджер** — сотрудник, обрабатывающий заказы.
4. **Администратор** — полный доступ.

## Связи «include» / «extend» / generalization

- **include**: UC6 (Оформить заказ) → UC5 (Авторизация)
- **include**: UC7 (Просмотр своих заказов) → UC5 (Авторизация)
- **extend**: UC2/UC3/UC4 (Поиск/Фильтр/Сортировка) → UC1 (Просмотр каталога)
- **generalization**: Администратор → Менеджер (админ выполняет всё то же + больше)
