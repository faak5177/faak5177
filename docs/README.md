## Что осталось сделать вручную

1. **Модуль 1 (DOCX + PDF)**: открыть `Модуль1_Резервное_копирование.md` в Word, вставить 5 скриншотов процесса бэкапа, сохранить DOCX + PDF.
2. **Модуль 2 (DOCX + PDF)**: открыть `Модуль2_Мониторинг_ресурсов.md`, вставить 4 скриншота (PRAGMA, свойства файла, SQL-отчёты), сохранить DOCX + PDF.
3. **Модуль 3 (PDF)**: открыть `UseCase_Соловьёв.drawio` на [app.diagrams.net](https://app.diagrams.net), File → Export as → PDF.
4. **Залить файлы** в `docs/` через GitHub веб-интерфейс (Add file → Upload files).
